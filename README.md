# VisionLens-Calibrator
> **OpenCV를 이용한 카메라 캘리브레이션 및 렌즈 왜곡 보정 도구**

본 프로젝트는 체스보드 패턴을 활용하여 광각 해늗폰 카메라의 내부 파라미터를 추정하고, 광각 렌즈 등에서 발생하는 이미지 왜곡을 수학적으로 보정하는 기능을 제공합니다.

## 📌 주요 기능
- **인터랙티브 이미지 선택**: 비디오에서 캘리브레이션에 적합한 프레임을 사용자가 직접 선택.(space와 Enter를 활용해서 영상을 멈추거나 정지된 이미지를 선택할 수 있습니다.)
- **카메라 파라미터 추정**: 초점 거리(f), 주점(c), 왜곡 계수(distortion coefficients) 산출.
- **실시간 왜곡 보정**: 산출된 파라미터를 적용하여 왜곡된 영상을 평면으로 보정(Rectification).

## 기능
### 1. Camera Calibration
- 비디오를 재생하며 다양한 각도에서 체스보드 프레임을 추출합니다.
- Space: 화면 일시정지 및 체스보드 프레임 추출
- Enter: 현재 프레임을 데이터셋으로 저장
- ESC: 선택완료 및 계산 시작

### 2. Distortion Correction
- 추출된 데이터 셋의 파라미터를 사용하여 영상을 보정
- Tab: 보정모드와 원본 모드를 전환합니다.
- ESC: 프로그램을 종료합니다.
